# vue-scaffold

Vue2 + Webpack2 + Sass + CSS + Vuex + VueRouter Scaffolding

A general usable template code for starting development using VueJS

##### Instructions

1. To install node dependencies:
  `npm install`
2. To run `development` build:
 `npm run build:dev`
3. To run `production` build:
 `npm run build:prod`
